# Instruction for codex

## essistial check before working

Common module-level utilities are stored in utils.py. Before starting work each time, you must check:

Whether utils.py contains sufficiently general-purpose functions. If so, move them to an appropriate location under ~/projects/myutils, and update PY_FILES_SUMMARY.md accordingly.
Review ~/projects/myutils/PY_FILES_SUMMARY.md to determine whether there are sufficiently useful functions that can be directly reused for the current task.

This check must not be removed and must be performed every time.
